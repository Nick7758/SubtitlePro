# NickSub Pro - AI 视频翻译专家


NickSub Pro 是一款基于AI技术的视频翻译软件，能够自动识别视频中的语音并将其翻译成多种语言，支持生成双语字幕和硬字幕视频。

## 功能特性

### 🎬 视频翻译
- 自动语音识别（ASR）支持多种语言
- 支持将视频内容翻译成多种目标语言
- 生成高质量的双语字幕文件（.srt/.ass格式）
- 可选择合成带硬字幕的视频文件

### 📥 视频下载
- 支持从多个平台下载视频（YouTube, Bilibili, Twitter等）
- 一键导入浏览器登录信息，方便下载需要登录的视频
- 内置FFmpeg支持，确保兼容性

### 💰 计费系统
- 按分钟计费的灵活付费模式
- 实时查看账户余额和使用情况
- 快速购买分钟数扩展使用额度

### ⚙️ 个性化设置
- 支持明暗两种主题模式
- 可自定义FFmpeg路径和工作目录
- 简洁直观的用户界面

## 技术架构

### 核心组件
- **Python 3.8+**: 主要编程语言
- **PyQt5**: 图形用户界面框架
- **FFmpeg**: 音视频处理工具
- **Requests**: HTTP请求库
- **yt-dlp**: 视频下载引擎

## 使用教程

### 视频翻译流程
1. 打开"视频翻译"标签页
2. 点击"选择视频…"按钮选择要翻译的视频文件
3. 选择源语言和目标语言
4. 根据需要选择是否合成硬字幕视频
5. 点击"开始任务"按钮启动翻译过程
6. 等待处理完成后，在结果目录中查看生成的字幕或视频文件

### 视频下载流程
1. 打开"视频下载"标签页
2. 在输入框中粘贴视频链接
3. 如需下载需要登录的视频，点击"一键导入登录"按钮
4. 点击"下载视频"按钮开始下载
5. 下载完成后可在指定目录找到视频文件

## 常见问题

### Q: 如何解决下载视频时出现的权限问题？
A: 对于需要登录的网站（如YouTube），请使用"一键导入登录"功能，程序会自动从您的浏览器中提取登录信息。

### Q: 翻译质量如何保证？
A: 我们采用先进的AI模型进行语音识别和翻译，对于专业术语和语境有良好的处理能力。如果遇到特殊领域内容，建议结合人工校对。

### Q: 如何查看我的使用额度？
A: 登录后，在界面右上角会显示您的账户信息和剩余分钟数。也可以在"购买分钟"标签页查看详细信息。
